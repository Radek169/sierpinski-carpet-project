# Frontend

Frontend aplikacji odpowiada za wizualizację Dywanu Sierpińskiego oraz interakcję użytkownika.

## Funkcjonalności
- rysowanie fraktala na canvas,
- animacja iteracyjnego wycinania,
- zmiana liczby iteracji,
- sterowanie animacją (Start, Stop, Reset).

## Technologie
- HTML
- CSS
- JavaScript
- Canvas API

## Uruchomienie
Otwórz plik `index.html` w przeglądarce lub użyj Live Server w VS Code.
