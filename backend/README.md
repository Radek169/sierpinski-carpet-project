# Backend

Backend projektu został przygotowany zgodnie z wymaganą strukturą aplikacji.

Na obecnym etapie pełni funkcję organizacyjną i może zostać rozszerzony o:
- obsługę konfiguracji aplikacji,
- zapis ustawień użytkownika,
- udostępnianie danych przez API.

## Technologie (planowane)
- Node.js
- Express

## Uwagi
W aktualnej wersji aplikacji główna logika znajduje się po stronie frontendu.
